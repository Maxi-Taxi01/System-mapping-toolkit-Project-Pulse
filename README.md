# Pulse — Systeemkaart Versie 2 (overzicht)

Statische overzichtspagina van de System Mapping Toolkit (fase 1–5) voor Project Pulse:
samenhang tussen informele & formele zorg in Rotterdam.

- `index.html` — volledig zelfstandige pagina (geen build nodig)
- Uitleg-modus (💡 knop rechtsonder): klik op een tekstdeel voor bron, onderbouwing en bijdrage

## Publiceren via GitHub Pages
1. Maak een lege repo op GitHub en push deze map:
   `git remote add origin https://github.com/<gebruiker>/<repo>.git && git push -u origin main`
2. Settings → Pages → Source: `main` branch, root.
3. De pagina staat dan op `https://<gebruiker>.github.io/<repo>/`.
